\begin{definition}
Differential field extension $E/F$ is a \textit{Picard-Vessiot extension}\index{Picard-Vessiot extension} if
\begin{enumerate}
\item $E,F$ have the same ring of constants
\item $E$ is by adjoining in solutions to a linear homogeneous differential equation
\end{enumerate}
\end{definition}
